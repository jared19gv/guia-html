CSS


Introducción a CSS: CSS (Cascading Style
Sheets) es un lenguaje utilizado para dar estilo a los elementos de un documento HTML. Permite

definir reglas de estilo que se aplican a cada elemento HTML en particular.
Sintaxis y Reglas de CSS: Las reglas de CSS se componen de un selector y una o varias declaraciones
que contienen propiedades y valores para modificar el estilo de los elementos HTML.
Selectores de CSS: CSS maneja diferentes tipos de selectores, como ID, clase, etiqueta, entre

, para aplicar estilos a elementos específicos.
Funcionamiento de CSS: CSS utiliza una sintaxis basada en reglas con selectores que conectan las
etiquetas HTML con los estilos definidos en la regla, y declaraciones que contienen propiedades y
valores.
Propiedades y Valores en CSS: CSS ofrece una amplia gama de propiedades para modificar estilos,
como fuentes, colores, tamaños, márgenes, entre otros.
Unidades de Medida en CSS: Las propiedades que afectan la posición, márgenes, anchura, altura,

etc., se pueden medir en píxeles, ems, porcentajes, centímetros y otras unidades similares.
Comentarios en CSS: Se pueden añadir comentarios en CSS entre /* y */ para documentar el código y
hacerlo más legible.
Aplicación de CSS al HTML: Para aplicar estilos CSS a un documento HTML, se vincula la plantilla
HTML a un archivo CSS para definir cómo se mostrará la información en la página web.
Diferencias entre HTML y CSS: HTML se utiliza para estructurar el contenido de un sitio web,

mientras que CSS añade propiedades visuales como forma, color, tamaño y márgenes a elementos HTML.
Modelo de Caja en CSS: El diseño en CSS se basa principalmente en el modelo de caja, donde los

elementos HTML se consideran cajas apiladas una sobre la otra, y CSS se utiliza para ajustar su tamaño,
color, posición, etc.

Existen tres tipos o formas de vincular el CSS a tu sitio web (HTML): externo (external), interno (internal) y en línea (inline).

– CSS externo. Es el CSS que existe como un archivo independiente y se enlaza a un documento HTML
mediante la etiqueta . Esta es la forma mas común para para añadir CSS a HTML, pues tan solo es
necesario cambiar ese documento para que el sitio web completo reciba los cambios deseados.

– CSS interno. El CSS interno es un código CSS incrustado dentro del documento HTM
Ventajas del CSS
Las ventajas que reporta el uso de hojas de estilo en cascada (CSS) son muchas, aquí tienes alguna de
las mas destacables:

Facilita realizar modificaciones visuales era todo un sitio desde un único lugar.
Permite mejorar el tiempo de carga.
Mejora la usabilidad web
Facilita la indexación de la web por parte de las arañas de indexación.
Simplifica disponer de versiones optimizadas para distintos objetivos.
Hace los cambios y la organización más sencilla.
